# Axal public route claim sources

Verified: 2026-07-16

This matrix records the current public sources used to reconcile the six Axal
preview routes. It supports customer-safe wording; it is not evidence of
production cutover, authenticated capability, statutory correctness, or a
specific customer configuration.

## Shared boundary

- Authentication, signup, sessions, tenant data, client data, documents,
  credentials, portal operations, and private infrastructure remain in the
  current Axal app.
- Axal supports workflow, evidence organization, and review posture. It does
  not replace CA, CS, legal, tax, or other professional judgement.
- Statutory requirements and portal facts must be checked against current
  official sources before client action.
- The Astro preview uses synthetic labels only and contains no production
  screenshot, customer artifact, taxpayer identifier, amount, or real deadline.
