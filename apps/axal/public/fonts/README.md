# Axal preview fonts

The Axal preview self-hosts the established public Axal type families without a
runtime Google Fonts request.

- Cormorant variable font and license:
  [google/fonts/ofl/cormorant](https://github.com/google/fonts/tree/main/ofl/cormorant)
- Outfit variable font and license:
  [google/fonts/ofl/outfit](https://github.com/google/fonts/tree/main/ofl/outfit)

Both directories contain the upstream `OFL.txt` distributed with the font.
The files were retrieved from the official Google Fonts repository on
2026-07-16.
